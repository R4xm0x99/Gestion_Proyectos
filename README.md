FinTech Secure – Plataforma de Gestión Financiera
📌 Descripción del proyecto
FinTech Secure es una aplicación web de gestión financiera desarrollada bajo metodologías ágiles (Scrum) y prácticas DevOps. Su objetivo es permitir a usuarios y pequeñas empresas gestionar sus finanzas de forma segura, integrando múltiples cuentas bancarias, generando alertas inteligentes y facilitando el cumplimiento fiscal.
El proyecto se desarrolla por el equipo TechOps Solutions, enfocado en la entrega incremental de valor mediante iteraciones cortas (sprints) y automatización del ciclo de desarrollo.

🎯 Objetivos

Desarrollar una plataforma segura de gestión financiera
Integrar múltiples entidades bancarias mediante APIs
Implementar autenticación multifactor (MFA)
Generar informes fiscales automáticos
Ofrecer alertas inteligentes al usuario
Aplicar prácticas DevOps para integración y despliegue continuo


⚙️ Metodología
El proyecto sigue un enfoque ágil basado en Scrum, con las siguientes características:

Iteraciones de 2 semanas (Sprints)
Product Backlog basado en historias de usuario
Priorización mediante técnica MoSCoW
Estimación con Planning Poker (story points)
Uso de Jira para gestión del proyecto
